# 公益法人会計-Excel

Excel / VBA で構築する公益法人会計システム。  
仕訳帳を基点に、各種財務諸表を自動生成する設計。

---

## 概要

本システムは、公益法人会計基準に基づき、
仕訳データから以下の帳票を自動生成することを目的とした
Excelベースの会計システムです。

年度単位でファイルを更新しながら運用する設計としています。

自分用に作ったもののため、未実装の部分は多いです。

---

## 対応機能

- 仕訳帳から以下を自動生成
  - 総勘定元帳
  - 貸借対照表
  - 正味財産増減計算書
- 補助科目に対応
- 勘定科目コードマスターを実装
- 補助科目コードマスターを実装
- 年度ごとにファイルを更新する運用設計

---

## 設計方針

- 仕訳帳を唯一の入力データとする
- コード体系（勘定科目・補助科目）を基盤とした集計構造
- Excel関数＋VBAによる自動集計
- 表示ロジックとデータ構造の分離を志向

---

## 現在の非対応事項

- 令和6年度 公益法人会計基準改定には未対応（対応予定）
- 指定正味財産の仕組みは未実装（現在開発中）
- 法人税
